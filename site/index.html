Kubernetes Context Aggregator

Discover, switch, and manage Kubernetes contexts from a unified interface — fast, local‑first, and privacy‑respecting.

Features

🔀

Context Switching

Switch active Kubernetes context with a single click and persist the change to disk automatically.

📂

Multi-Kubeconfig

Resolves KUBECONFIG env paths, merges multiple files, and picks the first file's current-context.

🖥️

Lens-like GUI

NavigationSplitView sidebar listing all contexts with a detail pane for Pods, Namespaces, and Deployments.

📌

Menu Bar Quick-Switch

Click the status-bar icon to switch context without opening the main window — perfect for fast workflows.

🔒

Secure Credentials

Bearer tokens and client certificates stored in the OS Keychain — never in plain text config files.

🚫

No Telemetry

Zero analytics, zero call-home, zero cloud accounts. Your cluster data stays on your machine.

Architecture

CubeLite is a monorepo with three independent layers — no runtime dependency between them.

🦀 Rust Core

crates/cubelite-core

Kubeconfig parsing, kube-rs client, resource types. Rust 1.82+, tokio, thiserror, anyhow.

🍎 macOS App

apps/macos

Native Swift 6 + SwiftUI menu-bar and main window. macOS 14+, Observation framework, direct URLSession to k8s API.

🖥️ Desktop App

apps/desktop

Cross-platform via Tauri v2 + Svelte 5 + TypeScript. shadcn-svelte, Tailwind v4, Vitest, Playwright.

Getting Started

Prerequisites

ToolVersion
Rust1.82+
Xcode16+ (macOS app)
Node.js20+
pnpm9+

Rust Core

cargo build --workspace
cargo test  --workspace

macOS App

xcodebuild build \
  -project apps/macos/cubelite/cubelite.xcodeproj \
  -scheme cubelite \
  -destination 'platform=macOS'

Or open in Xcode and press ⌘R.

Desktop App

pnpm install
pnpm --filter desktop dev
